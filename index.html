1. 이건 뭔가요?

AI(Claude, ChatGPT 등)에게 아래 프롬프트를 복사해서 보내면, 내 수업 내용으로 된 인터랙티브 보드게임이 자동으로 만들어집니다.

코딩 지식 없이도, AI가 보드판·퀴즈·효과음·애니메이션까지 전부 만들어줍니다.

3. 사용 방법 (3단계)

1
문제 준비하기
수업 내용으로 카테고리 2~5개, 카테고리당 10문제 내외 준비
보기 4개(또는 O/X), 정답 1개만 적으면 됨 — 이모지·색상은 AI가 알아서 배정!
💡 문제 만드는 3가지 방법 (편하신 방법으로 선택하세요)

방법 ① 직접 작성 — 카테고리 이름 + 문제를 직접 적기
방법 ② AI 자동 생성 — 카테고리 이름만 적고 "카테고리별 10문제씩 자동 생성해줘"
방법 ③ 교과서 PDF 첨부 — 교과서 PDF 파일을 함께 올리고 "이 교과서 내용 참고해서 문제 만들어줘"
　③은 Claude, ChatGPT 모두 파일 첨부 가능합니다. 교과서 통째로 넣으셔도 됩니다.
2
아래 프롬프트의 [📝] 부분만 수정 → AI에게 보내기
프롬프트를 통째로 복사 → [📝] 부분만 내 수업에 맞게 교체 → 전송
⚠️ "게임 시스템" 부분은 그대로 두시면 됩니다!
3
결과물 저장 & 실행
① AI가 만들어준 코드를 HTML 파일로 저장
　Claude → 아티팩트 우측 상단 다운로드(↓) 클릭
　ChatGPT → 코드 복사 → 메모장 붙여넣기 → "다른 이름으로 저장" → 파일명 게임.html
② 저장한 파일을 Chrome 브라우저로 열기
③ 마이크 사용 시 "허용" 클릭

+N점" (콤보 시 보너스도 표시) / 오답: "😊 아쉬워요! 정답은 N번" - "확인 ▶ 다음" 버튼 눌러야 다음 턴 진행 (자동 넘김 아님 — 교사 해설 시간 확보) [✈️ 자유여행 UI] - 팝업에 "어디로 갈까요?" + 칸 번호 선택 버튼 - 선택 후 말 이동 애니메이션, 도착 칸 효과 발동 [🎰 럭키 주사위 UI] - "주사위를 굴려 보너스 점수!" 팝업 → 굴리기 버튼 → 1~6 결과 → "+N점!" → 확인 [연출·효과음] - 말: 한 칸씩 CSS 트랜지션 + 스텝 효과음 - 상황별 내장 효과음 (Web Audio API, 외부 파일 없음) - 황금열쇠: 카드 뒤집기 CSS 애니메이션 + 두근두근 효과음 - 출발 통과: 반짝 이펙트 - 종료: 폭죽 파티클 + 팡파레 [디자인] - 배경: 밝은 하늘색 그라데이션 - 폰트: 굵게, 20px 이상 (가독성·저시력 배려) - 버튼: 크고 둥근, 터치 쉬운 사이즈 - 보드: 11×11 정사각형, 꼭지점 약간 크게 - 전체 한국어 [기술 요구사항] - 단일 HTML 파일 (외부 라이브러리 없음) - 반응형 (교사 PC + TV 출력, 태블릿) - 문제 카테고리별 랜덤 셔플, 풀 소진 시 재셔플 - requestAnimationFrame 이중 래핑으로 보드 렌더링 - 이벤트 콜백: 전역 변수 방식 (클로저 직렬화 회피) - cooldown 즉시 설정 (마이크 레이스 컨디션 방지) - 2바퀴+ 순환: · pos는 누적 정수, 보드 칸 = pos % 40 · 토큰 배치: getCellPx(pos % 40) · 칸 효과: CELLS[pos % 40].t · 39→0 경계에서 자연스럽게 이동 + 출발 보너스 처리 · 자유여행 시 현재 바퀴 기준 pos 재계산 ■ 제작 지시 위 스펙대로 완성도 높은 HTML 보드게임을 단일 파일로 만들어줘. 오류 점검을 반복하고 발견 즉시 수정해서 완벽 완성본을 줘.

예시 A: 직접 작성

■ 게임 정보 - 게임 제목: 💰 돈 계산 보드게임 - 수업 과목/단원: 수학 5단원 "화폐" - 대상: 특수학급 초등부 ■ 퀴즈 카테고리 및 문제 [카테고리1] 동전 퀴즈 / +1점 1. 100원짜리 동전 3개는 얼마? ①200원 ②300원 ③400원 ④500원 → 정답: ② 2. 가장 큰 동전은? ①10원 ②50원 ③100원 ④500원 → 정답: ④ 3. ... (10문제) [카테고리2] 지폐 퀴즈 / +1점 (10문제) [카테고리3] 물건사기 퀴즈 / +1점 (10문제) [카테고리4] 거스름돈 문제 / +2점 (10문제)

예시 B: AI 자동 생성 (카테고리만 적기)

카테고리 이름만 정해두시면, AI가 문제를 자동으로 생성합니다. 아래 프롬프트를 복사해서 게임 프롬프트의 "퀴즈 카테고리 및 문제" 칸에 붙여넣으세요:

[카테고리1] [📝 카테고리 이름] / +1점 [카테고리2] [📝 카테고리 이름] / +1점 [카테고리3] [📝 카테고리 이름] / +1점 [카테고리4 - 고배점] [📝 카테고리 이름] / +2점 위 카테고리에 맞춰 카테고리별 10문제씩 자동 생성해줘. 보기 4개, 정답 1개, 난이도는 [📝 쉽게/보통/어렵게], [📝 대상 학생 수준 설명] 수준.

예시 C: 교과서 PDF 첨부

교과서 PDF를 함께 올리시면, AI가 교과서 내용에서 문제를 추출하여 구성합니다. 아래 프롬프트를 게임 프롬프트의 "퀴즈 카테고리 및 문제" 칸에 붙여넣으세요:

(⬆️ 이 메시지에 교과서 PDF 파일을 함께 첨부합니다) 첨부한 교과서에서 [📝 해당 단원명, 예: 5단원 "화폐"] 내용을 참고해서 아래 카테고리별로 10문제씩 만들어줘. [카테고리1] [📝 카테고리 이름] / +1점 [카테고리2] [📝 카테고리 이름] / +1점 [카테고리3] [📝 카테고리 이름] / +1점 [카테고리4 - 고배점] [📝 카테고리 이름] / +2점 보기 4개, 정답 1개, 교과서 본문에 나오는 핵심 내용 위주로 출제. 난이도는 [📝 쉽게/보통/어렵게], [📝 대상 학생 수준 설명] 수준.

6. 자주 묻는 질문

Q. 카테고리를 2개만 해도 되나요?
네. 2~5개 자유롭게 설정하실 수 있습니다. 3개 이상이면 보드가 더 다양해집니다.
Q. 고배점 카테고리 꼭 있어야 하나요?
아니요. 전부 +1점으로 통일하셔도 무방합니다.
Q. AI가 코드를 잘못 만들면?
"오류 점검하고 수정해줘"라고 한 번 더 보내시면 됩니다.
Q. 마이크가 안 켜져요.
AI 미리보기 화면에서는 마이크가 차단됩니다.
HTML 파일 다운로드 → Chrome에서 직접 열기 → 마이크 허용하면 됩니다.
Q. TV에 연결했는데 글씨가 작아요.
Ctrl + +로 확대하거나, F11로 전체화면 전환하세요.
Q. 문제를 나중에 수정하고 싶어요.
AI에게 수정할 문제와 함께 "이 문제를 ○○로 바꿔줘"라고 보내시는 게 가장 편리합니다.
Q. 학생이 보드를 여러 바퀴 도나요?
네. 턴이 남아있는 한 계속 돕니다. 출발 칸을 지날 때마다 +1점, 퀴즈·황금열쇠도 매번 발동됩니다.
Q. 어떤 AI를 써야 하나요?
Claude(claude.ai) 추천. 아티팩트로 바로 미리보기 가능합니다.
ChatGPT(chatgpt.com)도 사용 가능합니다. 둘 다 PDF 첨부를 지원합니다.
Q. 교과서 PDF를 통째로 넣어도 되나요?
네. AI가 알아서 해당 단원에서 퀴즈 내용을 추출합니다. 특정 단원만 원하시면 "○단원 내용만 참고해줘"라고 한마디 추가해 주세요.
